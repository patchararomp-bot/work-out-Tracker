โปรแกรมนี้พัฒนาด้วยภาษา C# แบบ Windows Forms
ใช้สำหรับบันทึกข้อมูลการออกกำลังกาย เช่น รหัสท่า ชื่อท่า ระดับความหนัก และสถานะ

การทำงานของโปรแกรม
	•	ปุ่ม Add สำหรับเพิ่มข้อมูล
	•	ปุ่ม Clear สำหรับล้างข้อมูลในช่องกรอก
	•	ปุ่ม Exit สำหรับออกจากโปรแกรม

เครื่องมือที่ใช้
	•	Microsoft Visual Studio
	•	ภาษา C#
